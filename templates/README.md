# Templates

Templates de plano de testes e relatórios de avaliação, adaptados do meu uso profissional para o contexto de sistemas de IA.

## Planejado

- `plano-de-teste-para-feature-com-ia.md` — adaptação de um plano de teste tradicional considerando não-determinismo
- `relatorio-de-eval.md` — estrutura padrão para reportar resultados de avaliação de um sistema de IA (métricas, casos adversariais, recomendações)
- `quality-gate-para-agentes.md` — critérios de aceite antes de um agente ir para produção
