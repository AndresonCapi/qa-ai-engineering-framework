# Roadmap

Progresso da transição de Quality Engineering para Engenharia de IA Aplicada. Cada fase entrega pelo menos um artefato versionado neste repositório.

- [ ] **Fase 0 — Python "puro"**: reforço de Python fora do contexto de test runner (Robot Framework)
- [ ] **Fase 1 — Fundamentos de LLM**: glossário próprio em [`knowledge/`](./knowledge)
- [ ] **Fase 2 — Prompt Engineering**: biblioteca de prompts testados em [`prompts/`](./prompts), aplicados a geração/análise de casos de teste
- [ ] **Fase 3 — API Anthropic**: ferramenta de estruturação de bug reports via tool use em [`projects/`](./projects)
- [ ] **Fase 4 — RAG**: assistente de perguntas e respostas sobre documentação de testes em [`projects/`](./projects)
- [ ] **Fase 5 — Agentes**: agente que gera casos de teste + esqueleto de automação a partir de uma especificação, em [`agents/`](./agents)
- [ ] **Fase 6 — LLM Evals**: suíte de avaliação automatizada para o projeto de RAG, em [`playbooks/`](./playbooks) + [`workflows/`](./workflows)
- [ ] **Fase 7 — Claude Code no dia a dia**: registro de casos de uso reais aplicados à automação de testes, em [`docs/`](./docs)
- [ ] **Fase 8 — Portfólio**: curadoria contínua deste repositório e dos demais do perfil

**Última atualização:** julho de 2026
