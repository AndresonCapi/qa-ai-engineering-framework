# Skills

Módulos reutilizáveis de conhecimento aplicado — pequenos pacotes de "como fazer X bem", no espírito de skills usadas por agentes de IA (contexto reutilizável em vez de instrução recriada do zero a cada vez).

## Exemplo de formato

Cada skill é uma pasta com um `SKILL.md` descrevendo quando usar, e arquivos de apoio (templates, exemplos, checklists).

## Planejado

- `escrever-caso-de-teste-para-ia/` — como escrever casos de teste para sistemas não determinísticos
- `estruturar-prompt-de-avaliacao/` — como montar um prompt de LLM-as-judge consistente
